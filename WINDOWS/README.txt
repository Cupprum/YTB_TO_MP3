Linux(everything should be preinstalled)

required python2.7
pip
open terminal in your directory where you have setup.py, requirements.txt, app.py
type in terminal 'python setup.py'
when you'd like to use program type 'python app.py'

Windows

download python 2.7
link here: https://www.python.org/downloads/release/python-2714/
install it(it should also install pip)
dont forget to install packages like Tk/Tck
open cmd in right path
type in 'python setup.py'
or just doubleclick on setup.py
now everything should be prepared for downloading music from youtube :)
